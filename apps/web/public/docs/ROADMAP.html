Latest Done

✓
v10.2.6 Mini-Release
Critical user-reported bug fixes. OpenAI provider 400 fixed across all models. The chat completions API standardized on max_completion_tokens, and reasoning models (o1, o3, o4, gpt-5) need different parameter handling than earlier GPT models. Gemini tool calling now round-trips thoughtSignature so reasoning-enabled Gemini 2.5 Pro and Flash accept follow-up function-call turns. Sleep/wake recovery: the renderer no longer ends up on a blank screen with a client-side exception when the system resumes from suspend or unlocks. Capabilities awareness: Skales now answers "yes" when asked if it can do math, render HTML, copy code blocks, etc., reflecting features that were live since v10.2.5 but absent from the system prompt. Planner: weekly schedules honor the day-of-week selection (the previous default of Mon to Fri made weekly look indistinguishable from daily), defensive coercion in isTaskDue prevents silent type-mismatch bugs, and Planner task definitions now appear in the global Tasks list alongside one-off tasks. Provider audit pass: xai default model bumped to grok-3 (grok-2 retired). Auto-updater untouched. DNA invariants intact.

v10.2.5 — Polish & Fix
Stability and polish pass across nine sprint sessions. Critical: skill toggles now actually work — a latent bug was silently ignoring every toggle since the skills feature shipped. Playground per-mode overrides persist correctly across save (Server Action serialization fix). MCP state reporting returns real per-server status. Calendar Reminders endpoint reads skills.json correctly. Proxy Dispatcher routes through undiciFetch at all 12 provider sites. Cloudflare and NVIDIA token caps lifted to real 128K. New chat features: KaTeX math rendering, edit/branch/delete on user messages, manual compact button, token split tooltip, code block copy, Ctrl+A scoping inside code blocks and HTML preview. Polish: responsive header collapse below 1280px, provider switcher anchor fix, HTML preview copy button, Identity Maintenance toggle moved to its own settings section, Compact button hidden in Mini Mode, MCPClient default error sink. Auto-updater untouched. DNA invariants intact.
