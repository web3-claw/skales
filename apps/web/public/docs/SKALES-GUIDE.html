What is Skales?

Skales is a friendly AI agent that runs right on your computer. Think of it as your personal AI assistant - one that remembers you, connects to your apps, and acts on your behalf. It's private, powerful, and completely source-available under the Business Source License 1.1 (BSL).

Source-available, not open source. You can read and modify the code for personal use. Commercial use requires a separate license.

💻 Local & Private

Everything stays on your machine. Conversations, settings, API keys - none of it goes to any Skales server. Only your chosen AI provider sees your prompts.

🤖 Choose Your Brain

OpenRouter (free to start), OpenAI, Anthropic Claude, Google Gemini, Mistral, DeepSeek, xAI Grok, Groq, or Ollama for fully local AI. Switch anytime.

🧠 Bi-Temporal Memory

Skales learns about you over time. It automatically extracts memories from conversations and organizes them by time. The more you chat, the smarter it gets.

📱 Multi-Platform

Skales is a native desktop app with Telegram, WhatsApp, and Discord integration. Access from your phone via Tailscale - no cloud required.

🔧 Agent Tools

Skales can act - browse the web, read files, send emails, check your calendar, write and run scripts, manage tasks. With your approval for anything sensitive.

⚡ Execute Mode

Let Skales plan and execute multi-step tasks autonomously. You approve the plan first, then watch it work - with checkpoints you can pause or cancel anytime.

Quick Start (2 Minutes)

You don't need to be technical. No Terminal. No npm. Just download and install.

Requirements

  • A computer - Windows 10+ or macOS 12+
  • An API key from any LLM provider (we recommend OpenRouter - free to start)
  • That's it. No Node.js, no Terminal, no npm.

Installation

🪟 Windows

  1. Download Skales-Setup-6.1.1.exe from skales.app
  2. Double-click the installer
  3. Windows may show "Windows protected your PC" - this appears because Skales is not code-signed yet (code signing costs $200–400/year). Click More info → Run anyway. This is safe.
  4. Skales installs and creates a Desktop shortcut
  5. Launch Skales from the Desktop icon

🍏 macOS (Apple Silicon M1–M4)

  1. Download Skales-6.1.1-arm64.dmg from skales.app
  2. Open the DMG and drag Skales into Applications
  3. First launch: macOS will say "Skales can't be opened because it is from an unidentified developer"
  4. Fix: Right-click Skales in Applications → click Open → click Open in the dialog. This is only needed once.
  5. Why? Apple requires a $99/year Developer certificate for signed apps. As a solo indie developer, Skales doesn't have this yet.

Intel Mac: Download Skales-5.0.0.dmg instead - same steps.

🔒 macOS: Full Disk Access (if needed)

If Skales can't access certain folders:

  1. Open System Settings → Privacy & Security → Full Disk Access
  2. Click the lock icon (🔒) to unlock
  3. Click + and add Skales from Applications
  4. Restart Skales

Every Day After That

Launch Skales from your Desktop shortcut (Windows) or Applications folder (macOS). Skales minimizes to your system tray (🦎 icon) when you close the window - it keeps running in the background.

Something Wrong?

🔄 Troubleshooting

  1. Quick fix: Close Skales completely (right-click the 🦎 tray icon → Quit), then relaunch
  2. Data issues after import: Restart Skales - the app needs a fresh start to read imported data
  3. Nuclear option: Go to Settings → Danger Zone → Delete All Data. This removes ~/.skales-data entirely and restarts Skales with a fresh onboarding. Export your backup first!
  4. Reinstall: Uninstall Skales (Windows: Settings → Apps → Skales / macOS: drag to Trash), re-download from skales.app, install again. Your data in ~/.skales-data is preserved unless you manually delete it.

  2. Toggle on Desktop Buddy.
  3. A floating mascot appears in the corner of your screen. Click it to open the Spotlight input.
  4. Type your question or request and press Enter.

Tool Execution - The Buddy Can DO Things

In v6.0, the buddy graduated from a simple Q&A widget to a full action agent. When you ask it to do something, it uses the same tool pipeline as the main chat - write files, search the web, send emails, manage calendar events, and more.

⚡ Auto-Executed Tools

Safe, read-only actions run immediately with no prompt. Examples: web search, check weather, read a file, look up information. The result appears in the speech bubble.

✅ Approval-Required Tools

Actions with side effects pause and show Approve / Decline buttons directly inside the speech bubble. Examples: write a file, send an email, create a calendar event. You stay in control without opening the main window.

How the Approval Flow Works

  1. You type a request: "Send a reminder email to John about our 3pm call"
  2. The buddy processes the request and detects it needs to send an email.
  3. The speech bubble shows the action description + Approve and Decline buttons.
  4. Click Approve - the email is sent. The agent then continues the task if more steps are needed.
  5. Click Decline - nothing happens, the action is cancelled.

Buddy Conversations in Main Chat

Every buddy conversation is automatically appended to your active chat session in the main window. You can open the main chat at any time to see the full history of what the buddy did, including tool results and approval decisions.

⏳ Message Queue

When Skales is working on a task, new messages queue up instead of interrupting. Here's how it works:

How the Queue Works

  • Automatic: If Skales is busy, your message is placed in a queue automatically - you don't need to do anything
  • Visual indicator: A small "queued" badge appears on your message so you know it's waiting
  • Cancel button: A Cancel button appears next to queued messages. Click it to remove the message from the queue before it's processed
  • Order preserved: Messages are always processed in the order they were sent
  • Live updates: Once the current task finishes, the next queued message is picked up automatically - no need to resend

The queue is especially useful in Execute Mode, where tasks can take several minutes. You can keep sending follow-up instructions and they'll be processed in order.

🧠 Bi-Temporal Memory

Skales has a sophisticated memory system that works in two time dimensions - when things happened and when Skales learned about them.

🔵 Short-Term Memory

The current conversation. Skales always has context from your recent messages. This resets when you start a new session.

🟢 Long-Term Memory

Facts about you that persist forever. Your name, preferences, goals, recurring topics. Skales builds this automatically from your conversations.

📖 Episodic Memory

Specific moments and stories. "You mentioned your dog Biscuit in March." Skales stores significant events with timestamps.

⚡ Auto-Scan

The Memory Page

⚡ Execute Mode

Normal chat and Execute Mode are two different ways to interact with Skales.

💬 Normal Mode

Skales responds to one message at a time. It might use tools in a single step (search web, read file), but it always waits for you between each response. Best for conversations and quick tasks.

⚡ Execute Mode

For complex, multi-step tasks. Skales creates a plan, shows it to you for approval, then executes it step by step - searching the web, writing files, running commands, and more.

In Execute Mode:

  • Plan approval: Skales always shows you its plan before doing anything. You click Approve or Reject.
  • Checkpoints: At key moments, Skales pauses and shows you what it did. You can approve the next step or stop.
  • Approve/Reject in Telegram: If you're away from your computer, Skales sends the plan to your Telegram bot as an inline keyboard. Tap ✅ Approve or ❌ Reject directly from your phone.
  • Cancel anytime: Click the Stop button in chat at any time to halt the current execution.

✉️ Email (IMAP/SMTP)

Connect any standard email account - Gmail, Outlook, iCloud, Fastmail, ProtonMail, or any custom IMAP/SMTP server.

Gmail Setup

  1. Enable 2-Factor Authentication on your Google account (required for App Passwords)
  2. Go to myaccount.google.com/apppasswords and generate an App Password for "Mail"
  3. In Skales → Settings → Skills → Email, use these settings:
    • IMAP Host: imap.gmail.com · Port: 993 · TLS: On
    • SMTP Host: smtp.gmail.com · Port: 465 · TLS: On
    • Username: your Gmail address · Password: the App Password (NOT your Google password)
  4. Test the connection and save

Other Providers

📮 Outlook / Hotmail

  • IMAP: outlook.office365.com:993
  • SMTP: smtp.office365.com:587
  • Use your Microsoft account password or App Password

🍎 iCloud Mail

  • IMAP: imap.mail.me.com:993
  • SMTP: smtp.mail.me.com:587
  • Generate an App-Specific Password at appleid.apple.com

✉️ Fastmail

  • IMAP: imap.fastmail.com:993
  • SMTP: smtp.fastmail.com:465
  • Generate an App Password in Fastmail settings

🛡️ Custom IMAP

  • Check your email provider's support page for IMAP/SMTP settings
  • Standard ports: IMAP 993 (TLS) or 143 (STARTTLS)
  • SMTP: 465 (TLS) or 587 (STARTTLS)

What Skales Can Do With Email

  • List emails: Ask "What's in my inbox?" - no approval needed
  • Read emails: "Read the email from John" - no approval needed. HTML emails are automatically converted to readable text
  • Send email: Skales always asks for your approval before sending. You see the recipient, subject, and body before it goes out
  • Reply to email: Same as send - requires your approval
  • Move to Trash: "Delete that email" moves it to Trash (not permanent). Requires approval
  • Move to folder: "Move that to the Archive folder" - no approval needed (non-destructive)
  • Empty Trash: Requires explicit approval with a clear warning: "This will permanently delete all emails in Trash. This cannot be undone."

🦠 VirusTotal Attachment Scanning

When reading an email that has attachments, Skales shows a "Scan with VirusTotal" button next to each attachment name. This lets you check if an attachment is safe before you ever open it.

🔒 Security Rules for Email

  • Skales will NEVER auto-download or open attachments without your explicit approval
  • Skales will NEVER click or visit links in emails. If you ask about a link, Skales shows you the URL and says: "I cannot visit email links for security reasons. You can open this link in your browser."
  • These rules exist to protect you from phishing attacks and malware

👥 Group Chat

Group Chat is a unique feature where multiple AI personas hold a conversation with each other - and you can watch, join, or just read the transcript.

What it is

You define 3–5 AI participants, each with their own name, model, and persona. You set a topic or question. Skales orchestrates rounds of conversation between them - each participant responds to the previous one, building a real multi-perspective discussion.

How to Configure

  1. Go to Settings → Skills → Group Chat and enable it
  2. Go to Group Chat → Configure in the sidebar
  3. Add 3–5 participants. For each one, set: Name, AI Provider & Model, Persona description
  4. Set the conversation language and number of rounds (1–5)
  5. Save and go to Group Chat to start a session

💰 Cost Warning

Each round makes one API call per participant, plus a final summary call. Example: 3 participants × 3 rounds = 10 API calls. Use cheaper models (like Llama or Gemini Flash) for experimentation. Premium models for long sessions will add up quickly.

Export as Markdown

After a Group Chat session, click "Export as Markdown" to save the full conversation transcript as a .md file to your workspace.

🛡️ Website & Search Blacklists

Skales has a built-in security layer that prevents the AI from visiting dangerous websites or performing unsafe web searches - even if someone tricks it into trying.

🌐 Website Blacklist

A list of domains Skales will never visit, regardless of instructions. Pre-populated with known dangerous sites: paste sites (prompt injection risk), file sharing, chan boards, URL shorteners, exploit databases.

🔍 Search Safety Filter

A list of search terms Skales will refuse to search for. Pre-populated with hacking, doxxing, illegal activity, and prompt injection keywords.

How to Customize

  1. Go to Settings → Security → Security Blacklists
  2. Toggle each filter on or off independently
  3. Click the expand arrow to see the current list
  4. Type a domain (e.g. example.com) or search term and press Add or Enter
  5. Click ✕ next to any entry to remove it

Both filters are enabled by default. If Skales blocks something you legitimately need, just remove that entry from the list. Changes take effect immediately - no restart needed.

Critical Safety Feature

🔴 Killswitch

Skales is powerful. So powerful it needs an emergency brake.

The Killswitch is an emergency stop mechanism that immediately halts all Skales processes - no questions asked, no waiting. It is designed for situations where something has gone wrong, or you simply need everything to stop right now.

⚠ What the Killswitch Does

  • Immediately stops all running AI tasks, tool executions, and agent loops
  • Terminates any background processes started by Skales (scripts, commands, file operations)
  • Writes a log file to your Desktop: skales-killswitch-[timestamp].log with a full record of what was running
  • Shuts down the Skales server gracefully. You will need to restart it manually
  • No data is deleted - your conversations and settings are preserved

How to Trigger the Killswitch

🔴 Dashboard Button

In the Skales web interface, find the red "Killswitch" button in the Dashboard or Settings. Click it and confirm. Everything stops within seconds.

📱 Telegram Command

Send /killswitch to your Skales Telegram bot. This is the fastest way to stop Skales remotely - from your phone, anywhere in the world, even if the Skales app is unresponsive.

Auto-Triggers

Skales monitors itself and triggers the killswitch automatically if it detects dangerous conditions:

💾 RAM Overload

If Skales consumes an unusual amount of memory (sign of a runaway loop or memory leak), it stops itself before it can harm your system.

🔁 Infinite Loops

If the AI enters a loop that doesn't converge (detected by monitoring response patterns and iteration counts), execution is halted automatically.

📂 Unusual File Access

If Skales detects attempts to access sensitive system directories or files outside the designated workspace without authorization, it stops and logs the attempt.

After a Killswitch

Skales shuts down completely after a killswitch trigger. To restart:

  • Check the log file on your Desktop to understand what happened
  • Relaunch Skales from your Desktop shortcut (Windows) or Applications folder (macOS) to restart normally
  • If the issue persists: right-click the tray icon → Quit, wait a moment, then relaunch

Tips & Troubleshooting

✅ Best Practices

  • Export backups regularly: Settings → Export Backup saves everything as a ZIP. Do this before major changes or updates.

⚠️ Common Issues & Fixes

Skales won't start

  1. Windows: Right-click the Skales shortcut → Run as administrator (once)
  2. macOS: Right-click → Open → Open (Gatekeeper bypass, first time only)
  3. Check the 🦎 tray icon - Skales may already be running in the background
  4. If nothing works: uninstall, delete ~/.skales-data, reinstall for a clean start

Skales shows a blank or black screen

  1. Close Skales completely (Quit from tray icon), wait 5 seconds, relaunch
  2. If the problem persists: another app may be using port 3000 - Skales auto-detects ports 3000–3009 but a restart usually resolves this
  3. Last resort: uninstall, re-download from skales.app, reinstall

WhatsApp keeps disconnecting / QR code won't load

  1. Go to Settings → Skills → WhatsApp
  2. Click Disconnect (or Stop)
  3. Quit Skales completely (tray icon → Quit)
  4. Delete the WhatsApp session folder manually:
    Windows: C:\Users\YourName\.skales-data\integrations\whatsapp-session\
    macOS: ~/.skales-data/integrations/whatsapp-session/
  5. Relaunch Skales
  6. Go to Settings → Skills → WhatsApp → Start Setup
  7. A fresh QR code should appear - scan with your phone

WhatsApp won't connect on a new machine: If you imported a backup from another computer, the WhatsApp session is tied to the old machine. Delete the whatsapp-session folder (see above) and pair fresh.

API errors (401, 403, rate limit)

  1. Your API key might be wrong, expired, or out of credits
  2. Go to your provider's website and generate a new key
  3. In Settings → AI Providers, remove the old key and add the new one
  4. Check your account balance - many providers are pay-as-you-go

Voice input/output not working

  1. Check your microphone and speakers work in your OS settings
  2. Make sure your browser has microphone permission (check browser settings)
  3. In Skales Settings → TTS Configuration, check your TTS provider is configured
  4. Groq TTS requires a valid Groq API key

Telegram bot not responding

  1. Go to Settings → Skills → Telegram
  2. Verify your Bot Token is correct (get it from @BotFather on Telegram)
  3. Toggle Telegram OFF, wait 5 seconds, toggle ON
  4. If still not working: Quit Skales completely (tray icon → Quit), then relaunch
  5. The bot needs a full app restart after the initial token setup

Ollama not connecting

  1. Make sure Ollama is installed and running (check for the Ollama icon in your system tray)
  2. Check installed models: ollama list
  3. Pull a model if none: ollama pull llama3
  4. Ollama must be running on port 11434 (the default)
  5. In Skales → Settings → AI Providers → Ollama, confirm the base URL is http://localhost:11434

Frequently Asked Questions

🖼️ Images / Vision not working

You need an LLM that supports vision (image analysis). Not all models do. Examples that work: GPT-4o (OpenAI), Claude 3 Sonnet / Opus / Haiku (Anthropic), Gemini Pro Vision / 1.5 Pro (Google), LLaVA (Ollama). Check your provider's model list and switch to a vision-capable model in Settings → AI Providers.

📱 WhatsApp QR code not generating

Quit Skales completely (tray icon → Quit). Delete the WhatsApp session folder: Windows: C:\Users\YourName\.skales-data\integrations\whatsapp-session\ / macOS: ~/.skales-data/integrations/whatsapp-session/. Then relaunch Skales and go to Settings → Skills → WhatsApp → Start Setup. A fresh QR code will appear.

📡 How to access Skales remotely from my phone

Skales is a desktop app - the primary interface is the app itself. For mobile access, use Tailscale: install it on your PC and phone, then open your PC's Tailscale IP in your phone's browser (e.g. http://100.x.x.x:3000). On the same Wi-Fi, you can also use your PC's local IP. No VPS needed, no port forwarding, no public exposure. Everything is encrypted end-to-end.

📧 Email shows raw HTML

Skales now automatically converts HTML emails to readable plain text before showing them. If you still see HTML tags, try asking Skales to "re-read that email" - it will fetch and convert it again. If the issue persists, the email may use non-standard HTML encoding that the converter doesn't handle yet.

💰 Group Chat is expensive

Yes - each round makes one API call per participant, plus a final summary call. The math: 3 participants × 3 rounds = 10 API calls. 5 participants × 5 rounds = 26 API calls. For experimentation, use cheap models (Llama 3, Gemini Flash, Mistral 7B via OpenRouter). Save premium models for sessions where quality matters. You can also reduce rounds to 1–2 for a quick discussion.

🧠 How does Skales decide what to remember?

Every 90 minutes, Skales automatically scans recent conversations and extracts facts, preferences, names, recurring topics, and significant events. It uses your AI model to determine what's worth remembering. You can manually trigger this scan from the Memory page. You can also delete any memory Skales has saved - it won't use it again.

🔒 Is my data safe? Does Skales send my conversations anywhere?

Skales stores everything locally on your computer inside the .skales-data folder. No Skales server ever sees your data. The only exception: when you send a message to Skales, the text goes to whichever AI provider you configured (OpenAI, Anthropic, etc.). Skales itself never stores or transmits your conversations to any third party.

🛡️ Skales blocked a website I need to access

Go to Settings → Security → Security Blacklists, expand the Website Blacklist, find the blocked domain, and click ✕ to remove it. Changes take effect immediately. If you want to disable all domain checking temporarily, toggle off the "Website Blacklist" switch at the top of that section.

⚡ What's the difference between Execute Mode and normal chat?

Normal chat: one message, one response, Skales waits for you between each exchange. Execute Mode: you describe a complex goal, Skales creates a step-by-step plan, shows it to you for approval, then executes it autonomously - using tools, writing files, running searches, etc. - until the task is done or it hits a checkpoint requiring your input. Use normal mode for conversations, Execute Mode for multi-step projects.

📋 What is BSL 1.1? Can I use Skales commercially?

Business Source License 1.1 means Skales is source-available (you can read and modify the code), but not open source. Personal and non-commercial use is free. If you want to use Skales in a commercial product or service, you need a separate commercial license. After a certain period (specified in the license), it converts to an open source license. Check the LICENSE file in the Skales folder for the exact terms.

🔴 I triggered the Killswitch by accident. How do I restart?

Check the log file on your Desktop (skales-killswitch-[timestamp].log) to see what was running. Then relaunch Skales from your Desktop shortcut (Windows) or Applications folder (macOS). Your conversations and settings are completely intact - the killswitch only stops processes, never deletes data.

🛡️ Safety Mode

Safety Mode controls how much autonomy Skales has when executing actions. Configure it in Settings → Safety Mode. Since v6.0.1, Safety Mode uses two clear levels. In v6.0.2, Unrestricted mode correctly bypasses Telegram approval prompts as well.

Two Levels

ModeBehaviourUse Case
🟢 Safe (default)All actions require your explicit approval before execution. Skales shows you what it wants to do and waits for you to approve or decline.Daily use - recommended for most users
🔴 UnrestrictedSkales executes everything without asking. Full autonomy - you are responsible for reviewing what you ask it to do.Trusted automation pipelines, advanced scripting, power users

What Gets Blocked

  • rm -rf, del /f /s - recursive deletion
  • format C:, mkfs., dd if=... of=... - disk formatting
  • shutdown, reboot - system power commands
  • curl | bash - remote script execution
  • Fork bombs: :(){ :|:& };

Note: Safety Mode only applies to the execute_command tool. File read/write operations use the separate Workspace Security toggle in Settings.

📱 Mobile Usage & Tailscale

Skales is a desktop-first app - the primary interface is the native desktop application on your PC or Mac. Mobile access is a secondary method via your browser.

Accessing on Mobile (Secondary)

🏠 Same Wi-Fi

Open http://<your-pc-ip>:3000 in any mobile browser. Find your PC's IP in network settings or ask Skales: "What's my local IP?"

🌍 Remote via Tailscale

Install Tailscale on your PC and phone. Open http://<tailscale-ip>:3000 from anywhere. Encrypted, no port forwarding, no public exposure.

The UI is fully responsive - swipe-friendly sidebar, mobile input bar, and touch-optimized controls.

Known Limitations on HTTP

  • Clipboard copy: The browser Clipboard API requires HTTPS. On HTTP (LAN/Tailscale), Skales falls back to a textarea-based copy that works on most mobile browsers.
  • crypto.randomUUID(): Requires a secure context (HTTPS or localhost). On HTTP, Skales uses a Math.random-based fallback with equivalent entropy.
